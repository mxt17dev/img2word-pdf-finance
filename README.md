# 支付记录图片转 PDF 工具 (img2word-pdf-finance)

这是一个基于 Python 的自动化工具，专门用于将支付截图、收据图片整理成排版整齐的 PDF 文件。

## ✨ 功能亮点
* **自动排版**：支持一页 1 张或 2 张图片模式。
* **底部标注**：自动将图片原文件名（不含后缀）标注在 PDF 页面底部。
* **比例修正**：智能适配图片宽高比，防止拉伸变形。
* **中文支持**：完美支持中文文件名，不乱码。
